EFI-CGA

Guided and assisted a team of four in constructing a crucial customer-facing website. Throughout the project, I emphasized the promotion of robust software engineering principles, team cohesion, and individual growth, ultimately resulting in the delivery of a web application featuring clean code. In this project, we developed a web application that calculates the Frailty Index based on user input and stores the data for logged-in users.

  • Automated making a database backup action to ensure the security of our data. As a result our website was protected against SQL injection attacks or other attacks threatening our data.
  • Single handedly built the user authentication flow using industry preferred practices.
  • Used MVC design pattern to implement a website using .NET MVC framework which resulted in a structured, clean, and readable code.
  • Created a RESTful API in C# and tested it using the postman’s extension in google chrome.
  • Built the CI/CD build and release pipeline to improve and facilitate testing and keep the product owner up to date with the last changes that were deployed.
  • Used Entity Framework ORM for interacting with our database server through the code and as a result made changing, reading and reviewing the code easier.
  • Improved the performance of the site by eliminating the need for frequent page reloads using Single Page Application development practices.
